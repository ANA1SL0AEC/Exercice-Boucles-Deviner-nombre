# 🌟 Exercice sur les boucles – Deviner le Nombre 🌟

J'ai créé un nouveau petit programme pour m'entraîner à utiliser les boucles et les conditions sous Python ! 🎀

Il prend la forme d'un petit jeu qui consiste à deviner le nombre choisi au hasard par l'ordinateur.

---


## 🎯 Objectif du projet
Ce programme est un exercice pour **pratiquer les boucles (`while`) et les conditions (`if/else`)** en langage Python.  
L’utilisateur doit deviner un nombre choisi aléatoirement par l’ordinateur, compris entre 0 et 50. Puis le programme indique si son choix est trop grand ou trop petit jusqu’à ce qu’il trouve le bon nombre. ✨


🌈 Fun Fact : j'ai parfois eu du mal à tester ce programme dans mon terminal. Je tombais par hasard sur le bon nombre à la première tentative !!
